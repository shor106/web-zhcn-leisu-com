## 雷速平台是什么？

雷速（Leisu）是一个面向开发者和技术团队的高效协作平台，提供代码托管、持续集成、项目管理等功能。您可以通过访问 https://web-zhcn-leisu.com 获取雷速服务，该网站支持中文界面，便于国内用户快速上手。雷速旨在简化开发流程，提升团队生产力。

## 如何注册并开始使用雷速？

要使用雷速，请访问 https://web-zhcn-leisu.com 并点击“注册”按钮。填写邮箱、用户名和密码后，系统会发送验证邮件。完成验证后，您即可创建仓库、邀请团队成员并开始协作。雷速的注册流程简洁，无需额外配置即可体验核心功能。

## 雷速支持哪些代码托管和协作功能？

雷速提供完整的 Git 代码托管功能，包括仓库管理、分支保护、Pull Request 审核和代码审查。此外，在 https://web-zhcn-leisu.com 上，您还可以使用内置的 Issue 跟踪和看板视图来管理任务。雷速的协作功能支持实时通知和评论，帮助团队保持同步。

## 雷速的持续集成/持续部署（CI/CD）如何配置？

雷速内置 CI/CD 流水线，您可以在仓库设置中启用。通过编辑 `.leisu-ci.yml` 文件定义构建、测试和部署步骤。具体配置示例可在 https://web-zhcn-leisu.com 的文档中心找到。雷速的 CI/CD 支持多种语言和环境，并自动触发于代码推送或 Pull Request 事件。

## 雷速的安全性如何保障？

雷速采用传输层加密（TLS）保护数据，并支持双因素认证（2FA）增强账户安全。代码仓库默认私有，访问权限可精细控制。对于企业团队，雷速在 https://web-zhcn-leisu.com 提供审计日志和 IP 白名单功能。雷速定期进行安全审计，确保平台符合行业标准。
